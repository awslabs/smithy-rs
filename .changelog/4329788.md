Unnamed enums now validate assigned values and will raise a `ConstraintViolation` if an unknown variant is set.

The following is an example of an unnamed enum:
```smithy
@enum([
    { value: "MONDAY" },
    { value: "TUESDAY" }
])
string UnnamedDayOfWeek
```
